Banking Application — OOP Project

Console Java app (single-file example) demonstrating OOP concepts taught in your PPTs: accounts, FD, loans, credit & debit cards, transactions.

About the Project

This site documents the Java console Banking Application you built. The program uses only course topics: classes, inheritance, encapsulation, constructors, collections, interfaces, inner classes, Comparable/Comparator, arrays, and strings.

Features

  • Savings account — deposit, withdraw, transactions
  • Fixed Deposit (FD) — create FD and view maturity (simple interest)
  • Loan — apply and pay instalments, view outstanding
  • Debit & Credit cards — charge, pay, outstanding and minimum due (credit)
  • User register & login (in-memory)
  • Transaction history (per account) using an inner Transaction class

Run the Java program (locally)

  1. Save the Java file as BankApp.java.
  2. Compile: javac BankApp.java
  3. Run: java BankApp

Run online (Replit)

Create a new Replit (Java), paste BankApp.java, click Run. Share the Replit link with your instructor.

Technical Notes (match to syllabus)

This project strictly uses course topics (L1–L19 + inner classes + comparator/comparable):

  • Classes & Objects, Constructors, this, static, final
  • Inheritance, Polymorphism (overload/override), Abstract classes & Interfaces
